Chapter 6 Terminology

Terms Defintions
Machine Language Machine language refers to the set of instructions that a computer processor can understand and execute directly. Machine language instructions are represented in binary code and are specific to the hardware architecture of a particular computer system.
Debugging Debugging is the process of finding and fixing errors or defects in a computer program. Debugging tools are used to identify the location and cause of the error, and then the code is modified to remove the error.
Assembler An assembler is a program that converts assembly language code into machine language code.
Assembly Language Assembly language is a low-level programming language that uses mnemonics to represent machine language instructions. Assembly language is specific to the hardware architecture of a particular computer system.
Translator A translator is a program that converts source code written in one programming language to another programming language. A translator can be a compiler or an interpreter.
Compiler A compiler is a program that translates source code written in a high-level programming language into machine language code. The resulting code is then executed directly by the computer's processor.
Interpreter An interpreter is a program that translates and executes code one instruction at a time. Interpreters are commonly used for scripting languages and can be slower than compiled programs.
Cross-Platform Software Cross-platform software is software that can run on multiple operating systems or hardware platforms without modification.
Natural Language A natural language is a language that has evolved naturally over time, such as English, French, or Chinese. Natural languages are typically used for human communication.
Formal Language A formal language is a language that is designed for specific purposes, such as programming languages or mathematical notation. Formal languages have precise syntax and semantics.
Machine Independence Machine independence refers to the ability of software to run on different computer systems without modification.
High-Level Programming Language A high-level programming language is a programming language that is designed to be easy to read and write. High-level programming languages abstract away many of the details of the hardware, making it easier to write complex programs.
Object-Oriented Programming (OOP): Object-oriented programming is a programming paradigm that focuses on objects, rather than actions or functions. Objects have attributes and methods that define their behavior.
Scripting Language A scripting language is a programming language that is designed for quick prototyping or for automating tasks. Scripting languages are often interpreted and have simpler syntax than general-purpose programming languages.
Integer: An integer is a whole number, either positive or negative.
Float: A float is a decimal number.
String: A string is a sequence of characters.
Primitive Data Type: A primitive data type is a basic data type that is built into a programming language. Examples of primitive data types include integers, floats, and booleans.
Data Structure: A data structure is a way of organizing and storing data in a computer program. Common data structures include arrays, linked lists, and trees.
List: A list is a collection of values, ordered by index.
Tuple: A tuple is a collection of values, ordered and immutable.
Set: A set is a collection of unique values, unordered.
Dictionary: A dictionary is a collection of key-value pairs, unordered.
Function A function is a block of code that performs a specific task. Functions can take inputs and return outputs.
Local Variable A local variable is a variable that is defined within a function and is only accessible within that function.
Global Variable A global variable is a variable that is defined outside of any function and can be accessed and modified by any part of the program.
Source Program A source program is an original code written by a programmer in a high-level programming language.
Object Program An object program is the machine language code generated by a compiler from a source program.
