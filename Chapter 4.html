Chapter 4 Terms

Terms Defintions
Network Classifications
  • LAN (Local Area Network): A network that connects devices within a relatively small geographic area, such as a single building or campus.
  • MAN (Metropolitan Area Network): A network that connects devices within a larger geographic area, such as a city or region.
  • WAN (Wide Area Network): A network that connects devices over a large geographic area, such as multiple cities or even countries.
Network Models
  • Bus network: A network model in which all devices are connected to a single cable that runs the length of the network.
  • Star network: A network model in which all devices are connected to a central hub or switch.
Protocols:
  • CSMA/CD Ethernet: A protocol used for transmitting data over a LAN, in which devices listen for a clear signal on the network before transmitting data.
  • CSMA/CA Wifi: A protocol used for transmitting data over a wireless network, in which devices wait for a clear signal before transmitting data to avoid collisions.
Combining Networks
  • Repeater: A device that amplifies a signal to extend the range of a network.
  • Bridge: A device that connects two LANs to form a single network.
  • Switch: A device that connects devices within a LAN and directs traffic between them.
  • Router: A device that connects multiple networks and directs traffic between them.
  • Gateway: A device that connects two networks that use different protocols or technologies.
Client/Server Model
  • P2P (peer-to-peer): A model in which devices communicate directly with each other without the need for a central server.
  • Distributed system: A model in which multiple devices work together to perform a single task.
  • Cloud computing: A model in which data and applications are stored and processed remotely, rather than on a local device.
Internet Addressing
  • Tier 1: The top level of the internet's addressing system, which consists of a small number of networks that directly connect to each other.
  • Tier 2: The second level of the internet's addressing system, which consists of networks that connect to Tier 1 networks.
  • Tier 3: The third level of the internet's addressing system, which consists of networks that connect to Tier 2 networks.
IP Address
  • ICANN (Internet Corporation for Assigned Names and Numbers): The organization responsible for managing the internet's domain name system and IP address allocation.
  • DNS (Domain Name System): A system that translates domain names into IP addresses.
  • TLD (Top-level domain): The highest level of the domain name system, which identifies the type of organization or geographic location associated with a domain name.
  • Country-code TLDs: TLDs that are assigned to countries or territories, such as .uk for the United Kingdom or .cn for China.
Internet Protocols
  • FTP (File Transfer Protocol): A protocol used for transferring files over the internet.
  • HTTP (Hypertext Transfer Protocol): A protocol used for transmitting data over the internet, particularly for web browsing.
  • SMTP (Simple Mail Transfer Protocol): A protocol used for transmitting email messages over the internet.
World Wide Web
  • URL (Uniform Resource Locator): A string of characters that identifies the location of a resource on the web, such as a web page or image.
  • Client-side activities: Activities that occur on the user's device, such as running scripts or displaying images.
  • Server-side activities: Activities that occur on the web server, such as processing requests and accessing databases.
Internet four-layer software
  • Application layer: The layer of software that provides services to end-users, such as web browsing, email, and file transfer.
  • Transport layer: The layer of software that manages the transfer of data between applications on different devices, including protocols such as TCP and UDP.
  • Network layer: The layer of software that manages the routing of data between networks, including IP addressing and routing protocols.
  • Link layer: The layer of software that manages the transmission of data between devices on a single network, including protocols such as Ethernet and Wi-Fi.

Internet is
  • Packet-switching network: A type of network in which data is broken up into small packets and transmitted across the network individually, with each packet following its own route to the destination.
  • Fault-tolerant: The ability of a network to continue functioning even if some components fail or become unavailable.
  • Redundant: The presence of backup systems or components that can take over if the primary systems or components fail or become unavailable.
Cybersecurity
  • Malware: Software that is designed to damage or disrupt computer systems, including viruses, worms, and Trojans.
  • Virus: A type of malware that spreads by infecting other programs and files.
  • Worm: A type of malware that spreads by replicating itself and spreading to other devices on a network.
  • Trojan horse: A type of malware that is disguised as a legitimate program or file, but which actually contains malicious code.
  • Spyware: Software that is designed to monitor a user's computer activity and collect personal information without the user's knowledge or consent.
  • Phishing: A type of online scam in which attackers attempt to trick users into giving away personal information, often by posing as a legitimate organization or individual.
  • DDoS attack: A type of cyber attack in which a large number of devices are used to flood a target system or network with traffic, causing it to become overwhelmed and unable to function.
Protection
  • Firewall: A security system that monitors and controls incoming and outgoing network traffic, based on predetermined security rules.
  • Spam filter: A system that identifies and removes unwanted or unsolicited email messages.
  • Proxy server: A server that acts as an intermediary between client devices and other servers on the internet, often used for security or performance purposes.
  • Antivirus software: Software that is designed to detect and remove malware from computer systems.
  • Cryptography: The use of mathematical algorithms to secure communication and data storage.
  • Public key encryption: A method of encryption that uses two keys, one public and one private, to securely transmit information between devices.
